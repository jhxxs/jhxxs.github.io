重装win11和配置开发环境

July 18, 2022 · 3 min read
Kyle Von

为了拯救我的 C 盘，以及和 win11 dev 版说再见，一怒之下我重装了系统，随之而来的是一整天的瞎折腾，可谓是装系统十分钟，配环境大半天。

重装系统​

首先是重装系统，我选择的是经典的 U 盘启动盘方式：

1. 下载 win11 镜像​
2. 制作 USB 启动盘​
3. reboot & install​
4. 激活系统 🐙​

装机应用​

APPDesc

配置 nodejs 环境​

  • 安装nvm
# 安装 nvm, mac 或 linus 请直接google nvm，
# windows需要安装`nvm for windows`
# https://github.com/coreybutler/nvm-windows
scoop install nvm
# set nvm
nvm install lts     # mac 上是 nvm install --lts
nvm use lts         # 同理，mac 上是 nvm install --lts
# 查看是否启用成功
nvm list
  • 配置npm相关
~/.npmrc
registry=https://registry.npmmirror.com/
# proxy=http://127.0.0.1:7890/
